# Font Files

Place the Nuosu SIL font file here as `NuosuSIL-Regular.ttf`.

You can download Nuosu SIL from:
- SIL International: https://software.sil.org/nuosu/
- Or use the web fallback (Noto Serif Yi) if you don't have the local file

The app will automatically use the local font file if it exists, otherwise it will fall back to the web font.
